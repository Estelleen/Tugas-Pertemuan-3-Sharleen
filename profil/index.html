Tentang Saya

Saya lahir pada tanggal 2 April 2006. Saat ini saya berstatus sebagai mahasiswi Universitas Pendidikan Indonesia (UPI) dengan program studi Pendidikan Sistem dan Teknologi Informasi (PSTI). Saya dikenal sebagai seseorang yang menyukai warna ungu dan merah karena menurut saya kedua warna itu memiliki daya tarik tersendiri. Dalam hal cita-cita, saya memiliki impian unik: ingin menjadi ubur-ubur. Namun, bila impian itu tidak tercapai, saya berharap bisa mewujudkan cita-cita lain yaitu menjadi seorang dosen. Untuk urusan selera, saya cukup fleksibel, hanya saja saya tidak menyukai alpukat.
